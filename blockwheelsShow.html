BlockWheels

Inspired by Hotwheels, BlockWheels is a fully interactive JavaScript and HTML5 Canvas side-scroller game. This is a survival game, the longer the distance travel the greater the score. If you crash or fall off the tracks, the game is over. First project as a software engineer.

Lead Developer

  • Designed a dynamic 6-point hit box (even while rotating) for the car by incorporating JavaScript built-in trigonometric functions to track all four corners and 2 sides relative to cars central point and angle to ground, consistently outperforming canvas’ built-in collision function.
  • Created car rotation mechanic only while airborne by synchronizing canvas built-in image rotation vector with car rotation vector, in order to give the user more control over their landings.
  • Built a sliding functionality (by incrementing x and y positioning proportionately to the slope of given ramp) for the car when grounded in order to replicate the effect of wheels in real life and create an overall smoother user experience.

isBoundBy(){
  // bottom left, top left, top right, bottom right
  let angleOne = this.vector + this.theta  // bottom right angle
  let angleTwo = this.vector - this.theta // top right angle
  let angleThree = -this.vector + this.theta // bottom left angle
  let angleFour = -this.vector - this.theta // top left angle
  let angleFive = this.vector // right
  let angleSix = -this.vector // left
  let cornerArray = [angleOne, angleTwo, angleThree, angleFour]
  let sideArray = [angleFive, angleSix]
  let coordinateArray = []
  cornerArray.forEach ((angle, i) => {
      let x = this.hypotenuse * Math.cos(angle)
      let y = this.hypotenuse * Math.sin(angle)
      if (i < 2){
          coordinateArray.push([this.center[0] + x, this.center[1] + y])
      } else {
          coordinateArray.push([this.center[0] - x, this.center[1] + y])
      }
  })
  this.hitBox.bottomRight = coordinateArray[0]
  this.hitBox.topRight = coordinateArray[1]
  this.hitBox.bottomLeft = coordinateArray[2]
  this.hitBox.topLeft = coordinateArray[3]

  sideArray.forEach ((angle, i) => {
    let x = (CONSTANTS.CAR_WIDTH/2) * Math.cos(angle)
    let y = (CONSTANTS.CAR_WIDTH/2) * Math.sin(angle)
    if (i === 0){
        coordinateArray.push([this.center[0] + x, this.center[1] + y])
    } else {
        coordinateArray.push([this.center[0] - x, this.center[1] + y])
    }
  })
  this.hitBox.rightSide = coordinateArray[4]
  this.hitBox.leftSide = coordinateArray[5]
}

The above snippet is from the car class and defines it's hitbox at a given instance:

  • this.theta is the single angle needed to find the for relative corners of the rectangular "car" at any given moment if the vector of the car is known.
  • With each call, this function updates the car's hitbox property which contains six key value pairs (keys being the locations on the car, values being coordinates).
  • By utilizing a JavaScript Object (POJO) data structure to store the hitbox rather than an array, we can more easily differentiate between the "wheels" hitting the track and the "hood/trunk" hitting the track which would actually be game over.
